
# ❤️ Heart Disease Prediction App

## 📌 Overview

This project is a Machine Learning-based web application that predicts the likelihood of heart disease based on patient health parameters.
The application provides real-time predictions, personalized recommendations, and downloadable PDF reports.

---

## 🚀 Features

* 🔐 Login & Signup Authentication System
* 🧑 Patient Name Validation (only letters allowed)
* 📊 Heart Disease Risk Prediction
* 📈 Risk Score Visualization
* 🧾 Downloadable PDF Report
* 🕒 Prediction History with Date & Time
* 🎯 Dynamic Recommendations based on input
* 🌙 Dark Mode Compatible UI
* 🎨 Modern Login UI with Tabs (Sign In / Sign Up)

---

## 🧠 Machine Learning Model

* Algorithm Used: **XGBoost Classifier**
* Preprocessing: **StandardScaler**
* Accuracy: **~87%**

---

## 📊 Input Features

* Age
* Gender
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise-Induced Angina
* Oldpeak
* ST Slope

---

## 🛠️ Technologies Used

* **Python**
* **Streamlit**
* **Scikit-learn**
* **XGBoost**
* **NumPy / Pandas**
* **ReportLab (PDF Generation)**
* **HTML + CSS (UI Styling)**

## ▶️ How to Run the Project

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Bharati-M/heart-disease-prediction-app.git
cd heart-disease-prediction-app
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the App

```bash
streamlit run app.py
```
